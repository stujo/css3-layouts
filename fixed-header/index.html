position static

Normal positioning in the order of the document. top, right, bottom and left are ignored
Normal positioning means no overlapping siblings
static
(right: 10px; top: 20px;)

position relative

Allows the use of top, right, bottom, and left to position the element relative to it's normal position
relative
(right: 10px; top: 20px;)

position fixed

Fixed Relative to the viewport and doesn't scroll Allows the use of top, right, bottom, and left to position the element
fixed
(right: 10px; top: 20px;)
Look up in the top right of the page :)

position absolute

Like fixed except relative to the parent which is anything but 'static' position.
In this case the grey container has position: relative;
absolute
(right: 10px; top: 20px;)
